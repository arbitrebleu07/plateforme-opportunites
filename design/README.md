# Maquette Figma OpportuniTech

Fichier à importer :

```text
opportunitech-figma-mockups.svg
```

Dans Figma :

1. créer un nouveau fichier ;
2. glisser-déposer le fichier SVG dans la page ;
3. dégrouper les éléments si nécessaire ;
4. installer la police Inter ;
5. convertir chaque groupe nommé en Frame.

Écrans inclus :

- accueil desktop ;
- liste des opportunités desktop ;
- détail d'une opportunité desktop ;
- dashboard administrateur desktop ;
- accueil mobile ;
- liste des opportunités mobile ;
- connexion desktop et mobile ;
- inscription desktop et mobile ;
- confirmation de déconnexion desktop et mobile ;
- palette, typographie et principes du design.

Le SVG est vectoriel : les textes, couleurs, formes et espacements restent
modifiables après import dans Figma.
