# Diagrammes UML OpportuniTech

Ces fichiers PlantUML décrivent l'implémentation réelle de la plateforme.

- `cas-utilisation.puml` : rôles et fonctionnalités principales.
- `classes.puml` : entités métier et services techniques.
- `sequence-publication.puml` : publication d'une annonce par un membre.
- `sequence-scraping.puml` : collecte asynchrone déclenchée par un administrateur.

Ils peuvent être rendus avec l'extension **PlantUML** de Visual Studio Code ou sur un serveur PlantUML.
